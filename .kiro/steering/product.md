# Product Overview

Cursed Code Reviewer is an AI-powered code review system that analyzes pull requests and code files with a Halloween-themed, demonic personality.

## Core Features

- **Code Scanning**: Analyzes code for security, performance, and quality issues
- **AI Feedback**: Generates demonic-themed code review comments using AWS Bedrock (Claude)
- **Auto-Fix**: Generates patches to fix identified issues
- **Scan History**: Tracks all code scans and results
- **User Preferences**: Customizable severity levels and rule categories

## User Experience

- Halloween/demonic theme throughout (tombstones, crypts, souls, spectral analysis)
- Error messages and UI text use spooky terminology
- Dark theme with gothic aesthetics

## Key Terminology

- **SoulPool**: Cognito user pool (authentication)
- **SoulVault**: Authentication UI component
- **TombstoneDB**: DynamoDB table (data storage)
- **CodeCrypt**: S3 bucket (code storage)
- **CryptKeeper**: Authentication Lambda
- **SpectralAnalyzer**: Code scanning Lambda
- **DemonicOracle**: AI feedback Lambda using Bedrock
- **HauntedPatchForge**: Code fix generation Lambda
- **NightmareGateway**: API Gateway
